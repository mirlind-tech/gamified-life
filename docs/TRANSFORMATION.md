#### Gym Protocol (Baki Body Foundation)

**Goal:** Build visible muscle while cutting fat

**Split:** 4 days/week (Tue/Thu/Sat/Sun)
- Tuesday/Thursday: Upper body
- Saturday: Legs + Core
- Sunday: Weak points + Cardio

**Progression:**

| Phase | Weeks | Focus |
|-------|-------|-------|
| Linear | 1-4 | 3x5 compound lifts |
| Hypertrophy | 5-12 | 4x8-12 isolation |
| Peak | 13+ | Intensity techniques |

**Exercises:**
- **Chest:** Bench, Incline, Flyes (4x8-12)
- **Back:** Rows, Pulldowns, Deadlifts (4x8-12)
- **Legs:** Squats, Leg press, Lunges (4x8-12)
- **Arms:** Curls, Extensions (3x12-15)
- **Core:** Daily planks, leg raises

**Nutrition:**
- **Calories:** 2800-3000/day (maintenance + slight surplus)
- **Protein:** 150g/day minimum
- **Budget:** €255/month (€63.75/week)

**Sunday Prep:**
- 3 hours cooking
- 10 containers prepared
- 5 meals × 7 days planned
